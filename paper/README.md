# Paper Source

This folder contains the anonymous LaTeX submission source used for the manuscript, along with the exact figure PDFs referenced in the paper.

## Files

- `facesm-dc.tex`: main anonymous manuscript source.
- `cas-refs.bib`: bibliography database used by the manuscript.
- `cas-common.sty`: Elsevier support style file required for compilation.
- `cas-dc.cls`: Elsevier double-column class file.
- `cas-model2-names.bst`: bibliography style file used by the manuscript.
- `manifest.txt`: manifest file from the Elsevier template bundle.
- `fig1_facesm-flowchart.pdf`: overview figure for the FaceSM method.
- `fig2a_surrogate_breach.pdf`: surrogate-wise breach-rate figure.
- `fig2b_dataset_breach.pdf`: dataset-wise breach-rate figure.
- `fig3_lambda_sweep_vertical.pdf`: lambda-sensitivity figure.
- `fig4_embedding_trajectory.pdf`: embedding-trajectory analysis figure.
- `fig5a_sini_alignment.pdf`: vanilla gradient-alignment figure.
- `fig5b_alignment_gain.pdf`: alignment-gain figure under FaceSM.

## Notes

- The manuscript source is preserved in review-ready form.
- The figure PDFs here are the exact paper assets used in the anonymous submission.
- Raw dataset images and pretrained weights are not distributed in this repository.
