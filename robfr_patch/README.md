# RobFR Patch Files

This folder contains the local files used to adapt RobFR-style evaluation for the external validation experiments.

## Files

- `LGC.py`: local LGC attack implementation used in the patched RobFR workflow.
- `LGC_black.py`: black-box variant or support file for the LGC attack in the RobFR setting.
- `run_black_facesm.sh`: shell entry point previously used for the patched black-box RobFR run.
