# Core Files

This folder contains the main implementation files used by the study.

## Files

- `attack_plus.py`: main FaceSM attack logic and related evaluation helpers.
- `adv_cleanup_utils.py`: helper functions for organizing, filtering, or cleaning attack outputs.
- `ir152.py`: local model definition or loading support for the IR-152 victim model.
- `plot_pair_charts.py`: utility for plotting pairwise comparison charts from summary data.
- `sync_attack_performance.py`: helper for synchronizing or consolidating attack-performance records.
- `thresholds.json`: stored model- and dataset-specific threshold values used during evaluation.
